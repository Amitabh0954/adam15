# Cart Controller
This directory contains the controller logic for managing the cart functionality in the Adam15 project. The following controllers are included:

1. **Add to Cart**: Logic for adding items to the cart
2. **Remove from Cart**: Logic for removing items from the cart
3. **View Cart**: Logic for viewing the items in the cart
4. **Clear Cart**: Logic for clearing all items from the cart

Each controller is designed to handle HTTP requests and responses for the cart operations. Ensure to implement the necessary business logic and data validation for each controller.

# Analytics Tests

This directory contains tests for the analytics module of the backend.

## Directory Structure

- `unit_tests/`: Contains unit tests for individual functions and classes.
- `integration_tests/`: Contains integration tests that test multiple components working together.

## Running Tests

To run all tests, execute:

```sh
pytest
```

To run a specific test file, execute:

```sh
pytest path/to/test_file.py
```

To run a specific test within a file, execute:

```sh
pytest path/to/test_file.py::test_name
```
